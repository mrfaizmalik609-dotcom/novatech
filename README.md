# 💻 NovaTech - Laptop Store

NovaTech ek modern e-commerce website hai jo specially **Laptops & Tech Products** ke liye design ki gayi hai.  
Ye project fully responsive hai aur latest technologies ka use karta hai.

---

## 🚀 Features

- 🖥️ **Laptop Showcase** – Different categories of laptops with details  
- 🛒 **Cart & Buy Now** – Add to cart or directly purchase  
- 🔎 **Search Functionality** – Easily find laptops  
- ❤️ **Wishlist Option** – Save your favorite products  
- 🔐 **User Authentication** – Login / Signup with Firebase  
- 📱 **Fully Responsive** – Works perfectly on Mobile, Tablet & Desktop  
- 💳 **Checkout System** – With multiple payment options  

---

## 🛠️ Tech Stack

- **Frontend:** React.js + Tailwind CSS  
- **Routing:** React Router DOM  
- **State Management:** React Context API  
- **Backend & Auth:** Firebase  
- **Database:** Firestore
